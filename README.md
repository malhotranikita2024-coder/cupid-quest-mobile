## ✨ What this is

Cupid Quest is a browser-based 2D platformer designed around one idea — making small moments feel delightful.

Players navigate through themed levels, collect items, and complete challenges with increasing difficulty.

---

## 🧠 What I focused on

* Experience design over just gameplay
* Smooth interactions and feedback loops
* Emotional engagement through visuals and rewards

---

## ⚙️ Tech Stack

* React + Vite + TypeScript
* Supabase (authentication & backend)
* Lovable (no-code/low-code builder)

---

## 🎯 Key Features

* 7 themed levels with increasing difficulty
* Interactive collectibles and reward systems
* Leaderboard tracking
* Smooth animations and feedback loops

---

## 🚀 What this project shows

This project reflects how I build products:

* Starting from user experience
* Iterating based on feedback
* Focusing on details that improve feel, not just function
